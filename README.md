# Airline Database Management System

A SQL-based relational database designed to manage airline operations and maintain high data integrity. This project covers the full database lifecycle, from schema architecture to complex business reporting.

---

### Project Overview
This repository contains a complete database solution for the aviation industry, focusing on structured data storage and efficient retrieval.

* **Schema Design:** Full DDL scripts to create the database architecture.
* **Data Management:** Automated scripts for populating tables and managing data through Insert, Update, and Delete operations.
* **Business Intelligence:** SQL queries designed to extract meaningful insights from airline operations.

---

### Technical Components
* **CREATE_DB.sql:** Database structure and table definitions.
* **FILL_DB.sql:** Initial data population scripts.
* **DEL_UPD_INS.sql:** Scripts for data maintenance and transactional operations.
* **QUERIES.sql:** Analytical queries for business reporting and data retrieval.

---

### Tools Used
* **SQL / PostgreSQL**
* **Relational Data Modeling**
