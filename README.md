# SQL Database Management & Analysis System 📊

This project demonstrates a complete SQL workflow for managing and analyzing database systems. It covers everything from initial schema design to data manipulation and business reporting.

## Files Included
* **`CREATE_DB.sql`**: Defines the database structure (Tables, Primary Keys, Foreign Keys).
* **`FILL_DB.sql`**: Scripts for populating the database.
* **`QUERIES.sql`**: Advanced SQL queries for data analysis and reporting.
* **`DEL_UPD_INS.sql`**: Maintenance scripts for updating and cleaning data records.

## Key Skills Demonstrated
* Relational Database Design (RDBMS)
* SQL Joins & Aggregations
* Data Integrity & Constraints
* Database Maintenance & CRUD Operations
